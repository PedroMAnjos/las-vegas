# Painel Admin - Gestão & Financeiro
Sistema de administração web focado na gestão de mediadores e controlo financeiro semanal. Desenvolvido com uma interface moderna e sombria, otimizada para dispositivos móveis.

# Funcionalidades Principais
Autenticação Restrita: Sistema de login blindado para acesso exclusivo ao administrador.

Gestão de Mediadores:

Monitorização em tempo real de mediadores ativos, expirados ou em alerta.

Contagem decrescente automática de dias restantes com atualização diária de status.

# Módulo Financeiro:

Calculadora de entradas e saídas com categorias personalizáveis (Contratação, Renovação, Eventos, etc.).

Cálculo automático de saldo em caixa com indicadores visuais de lucro ou prejuízo.

Relatórios Semanais: Fechamento automático de caixa agrupado por semanas (Domingo a Sábado).

Persistência de Dados: Utiliza LocalStorage para manter as informações gravadas no navegador sem necessidade de um banco de dados externo.

# Interface e Design
Tema Dark: Estética inspirada em interfaces de alta tecnologia e gaming, utilizando cores como ouro primário, preto card e vermelho de alerta.

Responsividade: Design mobile-first com navegação inferior intuitiva e janelas modais para edição de dados.

Biblioteca de Ícones: Integração com Font Awesome para uma experiência visual rica.

# Tecnologias Utilizadas
HTML5: Estrutura semântica do painel.

CSS3: Estilização avançada com variáveis, Flexbox, Grid e animações de feedback visual.

JavaScript (ES6+): Lógica do sistema, manipulação de DOM e gestão de estado local.
